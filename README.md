# Python Logic Practice Programs

This repository contains beginner to intermediate level Python programs focused on strengthening logical thinking and problem-solving skills.

## Programs Included

1. Number Statistics (Largest, Smallest, Average)
2. Vowel, Consonant and Space Counter
3. Word Frequency Counter using Dictionary
4. Prime Number Checker with Factors
5. Continuous Number Analyzer with Exception Handling

## Skills Practiced

- Loops
- Conditional Statements
- Lists
- Dictionaries
- String Manipulation
- Exception Handling
- Input Validation
- Basic Optimization

## Goal

To build strong foundational logic for becoming a Python Developer.
